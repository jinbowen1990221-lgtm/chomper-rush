# Chomper Rush · 吃豆人大作战

移动端横屏复古港口吃豆竞技游戏原型。

## 在线试玩

**[打开《吃豆人大作战》在线版](https://jinbowen1990221-lgtm.github.io/chomper-rush/)**

建议使用手机横屏打开。首次进入时请允许浏览器播放声音。

这是一个使用原生 HTML、CSS 和 JavaScript 制作的多人竞技吃豆游戏 Demo。项目围绕“资源争夺、成长变强、反杀、冲榜”设计，适合在手机横屏环境中体验。

## 项目特色

- 移动端横屏适配，优先支持 iPhone Safari 横屏体验
- 手绘复古港口风格：首页、角色、模式入口和游戏 HUD
- 无尽模式与排位模式入口
- 虚拟摇杆和 BOOST 操作
- AI 玩家、港口守卫、金币、豆子、大能量豆和道具机制
- 排行榜、成长 HUD、结算页和设置页
- 皮肤资源与首页视觉素材

## 本地运行

进入项目目录后启动静态服务器：

```bash
python3 -m http.server 8793 --bind 0.0.0.0
```

电脑访问：

```text
http://127.0.0.1:8793/
```

手机访问时，确保手机和电脑在同一个 Wi-Fi，然后打开：

```text
http://你的电脑局域网IP:8793/
```
